Installation:-
    
    1. Open Vs code
    2. Click files icon
    3. Click on its last icon(Extension)
    4. On Extension search "Live Server",Install live server
    
You are Ready run your program i.e. "to convert your text to speech".

How it Works:- 
  1. The user types text into the text area.
  2. The available voices are fetched and displayed in the dropdown for selection.
  3. When the user clicks "Hear," the selected voice reads the text aloud using the Web Speech API's speech synthesis.


How To Use:-
   1) Enter your specified text in "Text Area"
   2) Select "voice"
   #) Click on translate.
